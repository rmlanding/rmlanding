A) HTML Structure - top

This template is a responsive layout using the Skeleton Framework. Skeleton's base grid is a variation of the 960 grid system. The syntax is simple and it's effective cross browser, but the awesome part is that it also has the flexibility to go mobile. The typography of Skeleton is designed to create a strong hierarchy with basic styles.

If you would like to edit the color, of any elements in one of these columns, you would do the following:

	a  { color: #747474; text-decoration: underline; outline: 0; text-decoration:none; }

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight. Alternatively you could do a find and replace for the color your using.

B) CSS Files and Structure - top

I'm using 6 CSS files in this theme. The first one is the main base.css file. Which includes the reset CSS. Many browser interpret the default behavior of html elements differently. By using a general reset CSS , we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

The font used are from google fontsis Arimo is used for all the body text. You can see samples of them here:
Arimo: http://www.google.com/webfonts/specimen/Arimo

If you wish to change the fonts used, simply find replace of the font you wish to use. eg replace with Arimo

h4 { font-size: 13px; line-height: 17px; margin-bottom: 16px; color:#333333; background-color:inherit; text-transform:uppercase; font-family:'Arimo', Helvetica, Arial, sans-serif; font-weight:bold; }

Within the header of each HTML find the following & replace with the google font you wish to reference
<link href='http://fonts.googleapis.com/css?family=Arimo' rel='stylesheet' type='text/css'>

C) JavaScript - top

This template imports a few Javascript files.

  1. general.js
  2. jquery-1.7.1.min.js
  3. jquery-ui-1.8.23.custom.min.js
  4. jquery.fitvids.js
  5. jquery.nivo.slider.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the other scripts. There are a few functions worth looking over. 
    general.js: To change the settings of open/close divs, toggles & buttons 
jquery.nivo.slider.js: To change the speed of the slider, change the different transition 
settings and edit the delay & speed
